Getting started

  1. Install Tampermonkey — the browser/mobile extension that runs userscripts. See the install guide →
  2. Pick a tool — each tool page has a one-click install button for desktop and mobile, per site.
  3. Open the site — Tampermonkey injects the script automatically on the matching URL.
Userscripts run client-side only. Use at your own risk — gamble responsibly.

Keno Preset Manager

Save and load Keno number + difficulty presets. No more clicking 10 tiles and re-picking risk every session.

What it does

  • Save your current Keno tile selection + difficulty (classic/low/medium/high) as a named preset
  • Load any preset with one click — tiles and risk set automatically
  • Presets shared across Stake sites via localStorage
  • Rename, reorder, and delete presets from the compact panel

How to use

  1. Install the script for your platform.
  2. Open the Keno game on Stake or Nuts. A preset panel appears near the board.
  3. Pick your tiles and risk manually the first time.
  4. Click "Save preset", give it a name (e.g. "Edge 5 Medium").
  5. Next time, click the preset name and the script selects all tiles + sets difficulty for you.

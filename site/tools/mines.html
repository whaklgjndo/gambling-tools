How to use

  1. Install the script for your platform.
  2. Open the Mines game. The helper GUI appears top-right (drag it anywhere).
  3. Set your mines count and bet in the Mines UI as normal.
  4. Set min/max picks and speed in the helper panel. Watch the next-gem probability update as conditions change.
  5. Click Start. The script will place bets, click random tiles, cash out, and repeat.
  6. Click Stop (or switch tabs) to halt between rounds.

Tips

  • High speed + high picks + high mine count = high variance. Start conservative.
  • The next-gem probability is exact given remaining tiles and mines — use it to calibrate your cashout target.

Tampermonkey Install Guide

The first install — the userscript host. Once Tampermonkey (or the iOS Userscripts app) is running, head to the Tool Install Guide for the actual unified script.

For Chrome, Edge, Brave, Opera, and any other Chromium-based browser. Firefox desktop also works — install Tampermonkey from addons.mozilla.org instead of the Chrome Web Store.

  1. Open Tampermonkey on the Chrome Web Store and click Add to Chrome.
    Open Chrome Web Store →
  2. Right-click the Tampermonkey icon near the address bar → select Manage extension.
  3. On the extension page, turn on “Allow User Scripts.” Chrome requires this — without it Tampermonkey refuses to inject any userscript.
  4. That's it for Tampermonkey. Head to the Tool Install Guide next to grab the unified script.

Pick your phone OS — the host is different on iOS vs Android.

iOS uses the free Userscripts app by Justin Wasack. It plugs into Safari as an extension and runs scripts just like Tampermonkey does on desktop.

  1. Open the App Store and search for Userscripts. Look for the </> icon by Justin Wasack. Tap GET.
  2. Open Settings → Apps → Safari → Extensions → Userscripts. (Search “Safari” in Settings if it's faster.)
  3. Toggle Allow Extension to ON. Under Other Websites, choose Allow.
  4. That's it for the host. Head to the Tool Install Guide → Mobile → iOS next.

Android Chrome doesn't support extensions. You need Firefox + Tampermonkey — the only browser/extension combo that works on Android.

  1. Install Firefox Browser by Mozilla from the Play Store. Not Firefox Focus, not Firefox Beta — only the regular Firefox supports extensions.
  2. Open Firefox → ⋮ menu → Extensions.
  3. Scroll to the bottom of the Extensions page and tap “Find more extensions.”
  4. Search “Tampermonkey,” open the first result, then tap Add to Firefox on the addons.mozilla.org page.
  5. Back in Firefox, open the ⋮ menu and tap the Tampermonkey icon near the top.
  6. That's it for the host. Head to the Tool Install Guide → Mobile → Android next.

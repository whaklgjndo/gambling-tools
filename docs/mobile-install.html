Tool Install Guide

The second install — the actual userscripts. Need to set up the host first? Tampermonkey Install Guide →

Mobile uses per-tool scripts. Grab the ones you want from the Tools Download Page, then follow the install steps for your phone OS below.

Tap the script link → Share → Save to Files → Userscripts folder.

  1. Open the Tools Download Page and tap the script you want. Safari opens the raw script (or pops the script icon).
  2. Tap the Share icon at the bottom → Save to Files.
  3. Pick the Userscripts folder as the save location.
  4. Tap Save (disk icon, top-right).
  5. Open Safari and visit the relevant site (Stake, Nuts.gg, etc.). Refresh once. The tool's UI appears.
  6. To enable / disable scripts at any time, tap the puzzle-piece icon in the URL bar and choose Userscripts.
  7. Repeat for each tool you want. Install only the scripts you actually need.

Long-press the script link → Copy → paste into Firefox's URL bar → tap INSTALL.

  1. Open the Tools Download Page in Firefox.
  2. Long-press the download link for the script you want, copy it, then paste it into Firefox's URL bar.
  3. Firefox opens a Tampermonkey install page — tap INSTALL.
  4. Open the relevant site (Stake, Nuts.gg, etc.) and refresh once. The tool's UI appears.
  5. Repeat for each tool you want. Install only the scripts you actually need.

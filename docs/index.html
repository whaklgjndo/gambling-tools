How it works

Three steps, then you're playing

The bundle is one userscript per platform. No build pipeline, no extension permissions, no account login — Tampermonkey injects it on the matching URL and the control panel does the rest.

Step 01

Install Tampermonkey

Free browser extension on desktop. iOS uses the Userscripts app; Android uses Firefox + Tampermonkey. The install guide walks each one.

Step 02

Add the unified script

One download. The install page has a single pinned link per platform — drag it into Tampermonkey on desktop, tap-install on mobile.

Step 03

Toggle tools on the page

Open Stake, Nuts.gg, or Shuffle. The ⚙ button appears bottom-left — flip the switches for the tools you actually want. The rest stay dormant.

Under the hood

Built like a tool, not a toy

⚡

One bundle, every site

Matches 14 domains (Stake, Stake.us, Stake mirrors, Nuts.gg + subdomains, Shuffle.com, Shuffle.us). Each tool only mounts on the games it supports.

🔌

Cross-tool integrations

The Dice Tool's Calculator / Optimizer / Stats stitch directly into the IOW/Smart HUD as an "Advanced IOW" tab on Stake and Shuffle Dice pages.

💾

Settings persist

Each tool saves its own configuration to localStorage. Keno presets are shared across sites; Auto-Vault tracks vaulted totals per currency per site.

🛟

Idle & stall aware

Rapid-fire bet engine auto-stops on stall (3s no settle), rapid-blocked (1.2s no Play button), and after every win for a 500ms cool-down so you actually see the result.

🎚️

Quick toggles per game

On games where a tool is active (Dice, Mines, Keno…), a small chip stacks above the ⚙ button so you can enable/disable that one tool without opening the panel.

🔒

Client-side, always

No telemetry, no remote calls, no auth. The script runs in your browser, talks only to the site you're already on. Open-source under your own copy of the repo.

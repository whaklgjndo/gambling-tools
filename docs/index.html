Getting started

  1. Install Tampermonkey (desktop) or the Userscripts app (iOS) / Firefox + Tampermonkey (Android). See the install guide →
  2. Install the unified script — one download covers every tool on every supported site. The install page has Desktop and Mobile builds pinned at the top of each tab.
  3. Open a supported site (Stake, Nuts.gg, or Shuffle) and refresh once. A small control panel appears — toggle the tools you want on, leave the rest off.
Userscripts run client-side only. Use at your own risk — gamble responsibly.

Five tools bundled into one userscript per platform. Pick a tab to see what's in the Desktop or Mobile build.

Everything below ships in Desktop/unified-desktop.user.js. One install in Tampermonkey, then toggle individual tools from the in-page control panel.

Watches your balance and sweeps a configurable percentage of profits to the vault. Big-win multiplier locks in more on spikes, rate-limited to 50 actions/hour.

Keno Presets

Save number selections and risk level as named presets, reload in one click. Presets are shared across Stake, Shuffle, and Nuts via a common storage key.

Keno only →

Mines Auto

Auto-plays Mines with weighted random tile picks in a Min/Max range. Live multiplier, payout, next-gem probability, and projected min/max payout. Auto-cashout when the board clears.

Stake Shuffle Nuts
Mines only →

Dice Tool

SWALD-style strategy builder. Calculator, parallel Web Worker grid-search optimizer, sortable results table, CSV export, one-click strategy import. Stitches into the IOW/Smart HUD as the "Advanced IOW" tab on Dice pages.

Mines Auto

Stake Shuffle Nuts Mines only

Set a Min/Max picks range and let it grind. Weighted random tile selection, live multiplier and next-gem probability, projected min/max payout range, and auto-cashout when the board clears. On Nuts it ships under the Hologlass-themed "Mines Burst" branding.

Games
Mines only
Picks per round
User-set Min/Max range (1–24)
Tile distribution
Geometric weight (base 1.5) — favors lower tiles
Poll cadence
15 ms loop, 600 ms per-tile timeout

Set Min and Max picks. Each round the bot rolls a random count in that range, clicks tiles one at a time (waiting for each to flip before the next), and auto-cashes when there are no idle tiles left. If it hits a mine, the round ends and the next bet fires.

Live stats show the Multiplier, current Payout, and exact Next Gem % based on how many mines and tiles are still on the board. The projected range card shows your guaranteed minimum and theoretical maximum payouts if you cashed now vs. cleared everything.

Engineering notes

  • Weighted tile pickerGeometric distribution with base 1.5 — biases toward lower-numbered tiles to mimic human picking patterns.
  • Click serializationOne tile at a time. Waits for data-game-tile-status to flip before the next click; 600 ms per-tile timeout prevents React stalls from hanging the engine.
  • Cashout logicAuto-cashes when there are no idle tiles left. On a mine hit the round just ends — the loss round triggers Play instantly.
  • Post-win cooldown500 ms pause after each successful cashout so you can see the final multiplier before the next round starts.
  • Button detectionReads [data-testid="bet-button"] / [data-testid="cashout-button"] for ready-state; respects disabled and aria-disabled.
  • Nuts (Hologlass)"Mines Burst" branding wrapper — same engine, tighter visual loop for the SPL-token denominated mines variant.

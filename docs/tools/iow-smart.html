The three modes

Pick the one that matches your playstyle. You can switch between them mid-session.

Mode 01

Manual

Static bet size every round. You set the wager once; the script just gives you the rapid-fire Start button and all the stats, leaving the bet math alone.

Mode 02

IOW (Increase on Win)

Bet grows by your Win Increase % on each win and resets to base after Loss Reset Streak consecutive losses. Optional win-reset count flashes the HUD when it triggers.

Mode 03

Smart

Bet size is calculated from Balance, target Multiplier, and an Aggression slider — auto-scales through five Gears as your streaks shift. Lock the gear to disable auto-scaling.

Core controls

  • Base BetStarting wager. Scales per currency (Nuts uses 0.00000001 SOL minimum).
  • Win Increase %IOW mode — multiplier applied to the bet after each win. Default 125 (= 1.25×).
  • Loss Reset StreakConsecutive losses before bet resets to base. Default 3.
  • Win Streak ResetIOW only — wins needed before auto-resetting on opposite result. Default 5.
  • Aggression sliderSmart mode — 0.1–2.0 scaling factor for streak-driven bet bumps.
  • History WindowSmart mode — lookback rounds for streak detection. Default 30.
  • Lock AggressionPin to a specific Gear (1–5) to bypass auto-scaling.
  • Stop Loss %Auto-stop if balance drops this % below session start.
  • Take Profit %Auto-stop if balance gains this % above session start.
  • Auto-Stop BalanceExact balance value at which to halt (optional).
  • Switch Over/UnderDice only — flip roll direction inline without touching the native control.
  • Start / Stop / ResetStart kicks off rapid-fire. Reset clears all session stats and restarts from current balance.

Rapid-fire engine

  • Poll cadence15 ms — matches React update speed without thrashing it.
  • Post-win cooldown500 ms pause after each cashout so you can actually see the result.
  • Rapid-blocked detectionIf the Play button doesn't reappear within 1.2 s, auto-stop.
  • Rapid-stall detection3 s with no bet settling auto-stops the engine (catches page freezes).
  • Auto-stop reasonsStop Loss / Take Profit / Auto-Stop Balance / stall / blocked — all logged in the HUD.

Per-site quirks

  • StakeFull-screen overlay; relocates the native sidebar into the top-right.
  • ShufflePurple themed (#6c47ff); custom dice-range slider added in the footer; sidebar collapsed (Shuffle has none by default).
  • NutsSimplified layout; SOL-denominated; min bet 0.00000001 SOL.
  • Dice ↔ Dice Tool stitchOn Stake/Shuffle Dice the Dice Tool's Calculator/Optimizer/Stats mount inside this HUD as an "Advanced IOW" tab — no second floating panel.
  • Click-sound muteShuffle — patches the Web Audio API to silence the 60 ms click sound on each Bet, leaving win sounds intact.

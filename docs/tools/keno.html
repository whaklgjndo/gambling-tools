Keno Presets

Stake Shuffle Nuts Keno only

Stop re-picking 10 tiles every round. Save your number set and risk level as a named preset, then load it back with one click — and your presets follow you across Stake, Shuffle, and Nuts.

Games
Keno only
Risk levels
Classic · Low · Medium · High
Preset storage
Shared across Stake, Shuffle, Nuts (same key)
UI
Draggable 260 px panel, bottom-right

Pick your tiles, set the risk dropdown, hit Save As…, give the preset a name. Done. The panel shows your current picks live, so you always know exactly what's on the board.

Next time you want that board: pick the preset from the dropdown, hit Load, and the script clicks the tiles for you with a 40 ms delay between clicks (so React stays in sync) and snaps the risk dropdown to the saved value.

Presets are shared. Save on Stake, load on Nuts. Save on Shuffle, load on Stake.us. Same names, same setups, every site.

Screenshots

Controls

  • Current picks displayShows pick count, risk level, and the comma-separated list of selected numbers (1–80).
  • Preset dropdownLists every saved preset as "Name (## picks, risk)."
  • LoadClicks the tiles in the saved order, toggles to reach the target set, snaps the risk dropdown. Shows a loading state during apply.
  • Save As…Prompts for a name. Stores numbers (1–80) + risk. Confirms before overwriting an existing name.
  • DeleteRemoves the selected preset after confirmation.
  • Draggable headerMove the panel anywhere; position persists between sessions on the same site.

How it tracks your picks

👆

Click capture

Captures tile clicks in the capture phase, so it sees every pick the user makes — independent of the game's own React state.

🔁

DOM re-sync

Polls data-selected="true" on tiles every 500 ms; a MutationObserver watches the grid for navigation changes.

🗂️

Single source of truth

The script's internal userPicks Set is authoritative, so the panel display matches reality even when the DOM lags.

Per-site quirks

  • Stake / ShuffleIdentical implementation; both use the native game-difficulty dropdown for risk.
  • NutsUses a 0–3 slider instead of a dropdown; the script maps the index to the same Classic/Low/Medium/High names so presets stay portable.
  • Cross-site presetsAll sites read/write the same localStorage keno-presets key — your library is genuinely shared.

Install Keno Presets

Ships in the unified bundle. Enable per-site from the control panel — the preset panel pops in at the bottom-right.

Dice Tool

On-site dice helper for Stake and Shuffle: win/loss counter, streak-based autoplay stopper, balance export, one-tap advanced strategy import.

What it does

  • Live win/loss counter with session stats
  • Streak-based autoplay stopper (halt on N consecutive losses)
  • Balance export — snapshot your balance to clipboard/JSON
  • One-tap import of an advanced strategy (preset chain rules)
  • Works on Stake Dice, Primedice, and Shuffle.us Dice

How to use

  1. Install the script for your platform.
  2. Open Stake Dice, Primedice, or Shuffle.us Dice. The helper panel appears.
  3. Set the max loss streak at which autoplay should halt.
  4. Start the site's autobet as normal. The counter tracks W/L in real time.
  5. Streak trigger: when you hit N losses in a row, the script stops autoplay automatically.
  6. Export / Import: use the export button to snapshot state, or the one-tap import to apply an advanced strategy preset.

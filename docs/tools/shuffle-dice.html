Dice Tool

Stake Shuffle Dice only (.com + .us)

Design your own SWALD-style Dice strategies with a Kelly-flavored calculator, grid-search the parameter space in parallel Web Workers, sort the results, then import the best one straight into the game with a single click.

Game

Web Workers (auto-detected, up to 4)

Inspired by SWALD's 8.47 dice strategy (StakeStats). The Dice Tool gives you everything you need to design, test, optimize, and run your own version of that style of strategy on Stake or Shuffle Original Dice — both .com and .us.

Three tabs: Calculator (enter one strategy, see all derived values), Optimizer (grid-search ranges in parallel Web Workers and rank the results), and Results (the optimizer's table, sortable, exportable to CSV).

On Stake and Shuffle Dice pages the tool's panel is stitched into the IOW/Smart HUD as an "Advanced IOW" tab — no second floating window.

Screenshots

Calculator tab

Enter one strategy. The tool shows every derived value and a single-strategy simulation.

  • BalanceBankroll to test. Default 20.
  • Win Increase %Bet multiplier applied on each win. Default 78.
  • Loss Reset StreakLosses before resetting to base. Default 5.
  • Bet DivisorBalance / divisor = base bet. Default 500.
  • Profit MultipleBase bet × this = per-cycle profit stop. Default 100.
  • Buffer %Safety margin for RNG variance. Default 25.
  • TrialsSimulations to run for the single-strategy case. Default 100.
  • Derived outputsBet Size · Profit Stop · Balance Target · fair Multiplier — all calculated live.
  • Import New StrategyWrites the current Calculator strategy into the game's native autobet UI.
  • Export Balance & Update StrategyReads your in-game balance and re-applies the strategy with the new bet size — for post-run resets.

Optimizer tab

Test every combination of parameter ranges in parallel and rank the results.

  • Starting BalancePreset list: 20, 50, 100 (or custom).
  • Trials per combo10 (quick) or 100 (full).
  • Bet Divisor256 or 500 (dropdown).
  • Profit Multiplier50 or 100.
  • Win %50–100, step 5.
  • Loss Streak3–5, step 1.
  • Buffer %25, 30, or 40.
  • Worker count1–4 (auto-detected from navigator.hardwareConcurrency, capped at 4).
  • Run OptimizerSpawns the workers, shows a progress bar, drops results into the Results tab.

Results tab

Every optimization run lands here. Sortable, exportable, re-importable.

  • ColumnsStarting Balance · Trials · Bet Div · Profit Mult · Win % · Loss · Buffer % · Avg High · Std Dev · Max High · Avg Cycles · Avg Rounds · Cycle Success % · Bust % · Score.
  • ScoreKelly-like metric: (Avg High − Starting Balance) / Std Dev. Default sort, descending.
  • Sort by any columnClick the column header to re-sort.
  • Export CSVDownload the full results history for spreadsheet work.
  • Import StrategyLoad a row's parameters back into the Calculator.
  • Keep PreviousOptional toggle — append new runs instead of replacing the table.

Engineering notes

🧪

Provably-fair RNG

The Web Worker runs Stake's verified HMAC-SHA256 RNG — the same algorithm used by the actual game — so simulations match real behavior.

⚙️

Compounded trials

Each trial runs cycles (profit targets) until bust or success, tracking peak balance, cycle count, and rounds.

🧮

Grid-search optimizer

Generates every parameter combination, distributes them across Web Workers, then aggregates avg/std/max peaks, cycle success %, and bust rate.

💾

Persistent results

All optimizer results saved in dice_tool_aio_desktop_state_v1 (localStorage) and re-loadable across sessions. CSV export for backup.

🎨

Themes

Original (dark cyan), Stake (dark blue), Shuffle (purple). Colors adapt to the host site automatically.

🔢

Streak Counter widget

Floating counter shows current win/loss streak, with a Target value and an audio chime when hit. Optional auto-stop when target reached.

Stats tab (Dice pages only)

On Stake and Shuffle Dice pages the Dice Tool's Stats tab subscribes to the IOW/Smart HUD via a public window callback (__iow_smart_on_publish__) and renders session balance, profit, peak, and streak data without doing its own bookkeeping. One source of truth, real-time updates per bet — not on a 500 ms tick.

Supports Stake.com, Stake.us, Shuffle.com, and Shuffle.us — no separate Shuffle build needed.

Install Dice Tool

Ships in the unified bundle. Enable the matching Stake Dice or Shuffle Dice tool from the control panel — the Dice Tool stitches itself into the IOW/Smart HUD as the "Advanced IOW" tab.

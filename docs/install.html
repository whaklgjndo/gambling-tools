Tool Install Guide

The second install — the actual unified script. Need to set up the userscript host first? Tampermonkey Install Guide →

One download. Drag it into the Tampermonkey editor. Done.

  1. Download the unified userscript.
    Desktop Download
    Unified Desktop — all five tools Stake Shuffle Nuts
    Auto-Vault, IOW/Smart, Keno Presets, Mines Auto, and the Dice Tool, bundled with an in-page control panel.
  2. Click the Tampermonkey icon → Create a new script…
  3. Drag the downloaded .user.js from your file explorer into the editor.
  4. Click Install on the confirmation page that appears.
  5. Open Stake, Nuts.gg, or Shuffle and refresh once. The ⚙ control panel appears in the bottom-left — flip on the tools you want.

Pick your phone OS — the install gesture is different on iOS vs Android.

Tap the link → Share → Save to Files → Userscripts folder.

  1. Tap the download link below. Safari opens the raw script (or pops the script icon).
    Mobile Download
    Unified Mobile — all five tools Stake Shuffle Nuts
    Same bundle as Desktop, with mobile UI shims for the iOS Userscripts app.
  2. Tap the Share icon at the bottom → Save to Files.
  3. Pick the Userscripts folder as the save location.
  4. Tap Save (disk icon, top-right).
  5. Open Safari and visit a supported site. Refresh once. The ⚙ control panel appears in the bottom-left.
  6. To enable / disable scripts at any time, tap the puzzle-piece icon in the URL bar and choose Userscripts.

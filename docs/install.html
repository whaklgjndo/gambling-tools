Tool Install Guide

The second install — the actual unified script. Need to set up the userscript host first? Tampermonkey Install Guide →

One download. Drag it into the Tampermonkey editor. Done.

  1. Download the unified userscript.
    Desktop Download
    Unified Desktop — all five tools Stake Shuffle Nuts
    Auto-Vault, IOW/Smart, Keno Presets, Mines Auto, and the Dice Tool, bundled with an in-page control panel.
  2. Click the Tampermonkey icon → Create a new script…
  3. Drag the downloaded .user.js from your file explorer into the editor.
  4. Click Install on the confirmation page that appears.
  5. Open Stake, Nuts.gg, or Shuffle and refresh once. The ⚙ control panel appears in the bottom-left — flip on the tools you want.

Install Tampermonkey

Install the userscript manager for your platform once. Then open any tool page and click an install button — you'll get prompted to install the script.

Works on Chrome, Edge, Brave, Opera, Firefox, and Safari.
  1. Install Tampermonkey — go to tampermonkey.net and click Download for your browser. Add the extension from your browser's store and pin it to the toolbar.
  2. Right-click the Tampermonkey icon → Manage extension. This opens the extension's settings page where you can enable user scripts.
  3. Enable "Allow User Scripts" on the extension page (required on Chrome/Edge/Brave on current versions — replaces the old Developer Mode toggle).
  4. Click the Tampermonkey icon in the toolbar — the popup shows your scripts and options like Create a new script and Find new scripts.
  5. Install a tool — on any tool page on this site, click the Install for Desktop button. Tampermonkey will show a confirmation page with the script name and matched URLs. Click Install.
Manual install (fallback): if a link ever opens as raw code, open Tampermonkey's dashboard, create a new empty script, and drag the .user.js file into the editor.
Regular Chrome on Android does not support extensions. Use Kiwi Browser (a Chromium-based browser that supports Chrome Web Store extensions) — or any other Chromium-on-Android browser with extension support.
  1. Install Kiwi Browser from the Play Store. Open it, go to the Chrome Web Store, and search for Tampermonkey. Tap the Tampermonkey entry to add it to Kiwi.
  2. Open the browser menu (⋮) → Extensions to confirm Tampermonkey is enabled and to open its dashboard.
  3. Enable Developer Mode in the extensions page if prompted (required for user scripts on newer Chromium versions).
  4. Install a tool — open any tool page on this site in Kiwi, tap Install for Mobile, and confirm the Tampermonkey install prompt.
On iOS, userscripts run inside Safari via the free, open-source Userscripts app by Justin Wasack.
  1. Open the App Store, search "userscripts", and install Userscripts by Justin Wasack (icon: </> on a light grey tile).
  2. Open the Settings app and search for Safari.
  3. Scroll to General → Extensions inside Safari settings.
  4. Tap Userscripts and turn it On in the extensions list.
  5. Open the Userscripts extension page and enable Allow Extension. Grant website access — set Other Websites to Allow so scripts can run on Stake and Nuts.
  6. Back in Safari, tap an install button on any tool page on this site. Safari will show the .user.js file — tap the Share icon at the bottom.
  7. Pick the Userscripts folder from the save dialog — tap the dropdown and choose Userscripts (the folder the extension created).
  8. Tap Save in the top-right.
  9. Open the game page (e.g. stake.us/casino/games/keno). Tap the Userscripts icon in Safari's address bar to open the popover and confirm the script is listed and enabled.
  10. Done. The script now runs every time you open a matching page. Toggle individual scripts on/off from the Userscripts popover.

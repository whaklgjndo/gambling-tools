Install Tampermonkey

Tampermonkey runs these userscripts in your browser. Install it once, then open any tool page and click the install button. Pick your platform below.

Works on Chrome, Edge, Brave, Firefox, Opera, and Safari.
  1. Go to the Tampermonkey site and click Download for your browser — tampermonkey.net.
  2. Add the extension from your browser's store and confirm the install prompt.
  3. Pin the Tampermonkey icon to your toolbar (recommended) so you can manage scripts easily.
  4. Enable Developer Mode in your browser's extensions page (Chrome/Edge) — required for userscripts to run reliably.
  5. Install a tool — open any tool page on this site, click Install for Desktop, and confirm the Tampermonkey prompt.
If an install link opens as raw code instead of showing a prompt, make sure Tampermonkey is enabled and the URL ends in .user.js.
On Android, Tampermonkey works inside Firefox for Android. Chrome on Android does not support extensions.
  1. Install Firefox for Android from the Play Store.
  2. Open Firefox → menu → Add-ons, then search for and install Tampermonkey.
  3. Enable the add-on when prompted and grant permissions.
  4. Install a tool — open any tool page on this site in Firefox, tap Install for Mobile, and confirm the Tampermonkey prompt.
On iOS, userscripts run inside Safari via the Userscripts app (a free, open-source Safari extension).
  1. Install the Userscripts app from the App Store (search “Userscripts” by Justin Wasack).
  2. Open Settings → Safari → Extensions and enable Userscripts.
  3. Grant access to All Websites (or at least to stake.com / stake.us / nuts.com) so scripts can run.
  4. Open the Userscripts app and pick a folder for scripts (create one if prompted).
  5. Open a tool page on this site in Safari, tap Install for Mobile. The Userscripts extension will prompt you to save the script.
  6. Enable the script from the Userscripts popover in Safari (tap the aA menu → Userscripts icon).

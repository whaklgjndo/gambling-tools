Install guide

Get the unified bundle running

Pick your platform below. Each tab has the one download link and the exact steps — Chrome & Edge on desktop, Firefox + Tampermonkey on Android, the Userscripts app on iPhone.

You'll install Tampermonkey (the userscript host) once, then drop in the unified userscript that bundles every tool.

  1. Install Tampermonkey from the Chrome Web Store.
    Open Chrome Web Store →

    Works the same on Edge, Brave, Opera, and other Chromium browsers.

  2. Right-click the Tampermonkey icon near your address bar and select Manage extension.
  3. On the extension page, turn on “Allow User Scripts.” Chrome requires this for any userscript to run — Tampermonkey itself will refuse to inject without it.
  4. Download the unified userscript.

  5. Click the Tampermonkey icon → Create a new script…
  6. Drag the downloaded .user.js from your file explorer into the editor.
  7. Click Install on the confirmation page that appears.
  8. Open Stake, Nuts.gg, or Shuffle and refresh once. A green ⚙ button appears in the bottom-left corner — that's the unified control panel.

After install

Click the ⚙ button to open the panel. Tools are grouped by site (Stake / Shuffle / Nuts). Flip the switch on each tool you want to use — disabled tools don't render or run.

For tools that hijack the game UI (IOW/Smart on Dice and Limbo), the page reloads when you toggle them so they can wire themselves into the right place.

Android Chrome doesn't support extensions. You'll need Firefox + Tampermonkey — the only browser/extension combo that works on Android.

  1. Install Firefox Browser by Mozilla from the Play Store. Not Firefox Focus, not Firefox Beta — only the regular Firefox supports extensions.
  2. Open Firefox → ⋮ menu → Extensions.
  3. Scroll to the bottom of the Extensions page and tap “Find more extensions.”
  4. Search “Tampermonkey,” open the first result, then tap Add to Firefox.
  5. Back in Firefox, open the ⋮ menu and tap the Tampermonkey icon near the top.
  6. You're now in Tampermonkey's dashboard. Installed Userscripts, Settings, and Utilities live here.
  7. Long-press the download link below, copy it, then paste it into Firefox's URL bar.

On iOS the host is the free Userscripts app (Justin Wasack). It plugs into Safari as an extension and runs scripts the same way Tampermonkey does on desktop.

  1. Open the App Store and search for Userscripts. Look for the </> icon by Justin Wasack. Tap GET.
  2. Open Settings → Apps → Safari → Extensions → Userscripts. (Searching “Safari” in Settings is the fastest way in.)
  3. Toggle Allow Extension to ON. Under Other Websites, choose Allow.
  4. Tap the download link below. Safari shows the raw script (or pops the script icon).

Install Guide + Tool Downloads

Pick your platform. Every tab has step-by-step instructions and a pinned download list with all the tools.

Before using any tools, you must install Tampermonkey.

  1. Go to Tampermonkey's site and click Add to Chrome.
    Open Chrome Web Store →
  2. Right-click the Tampermonkey icon near your address bar → select Manage extension.
  3. On the extension page, find Allow User Scripts and turn it on. This is required by Chrome — without it, no userscript can run.
  4. Download a tool from the Desktop Downloads pin.

Before using any tools, you must install Firefox + Tampermonkey. Chrome on Android doesn't support extensions.

  1. Install Firefox Browser by Mozilla from the Play Store. Not Firefox Focus, not Firefox Beta — only the regular Firefox supports extensions.
  2. Open Firefox → tap the ⋮ menu → Extensions.
  3. On the Extensions page, scroll to the bottom, and tap Find more extensions.
  4. Search for Tampermonkey, tap the first result, then tap Add to Firefox on the addons.mozilla.org page that opens.
  5. Back in Firefox, open the ⋮ menu → tap the Tampermonkey icon near the top.
  6. This opens Tampermonkey's dashboard with Installed Userscripts / Settings / Utilities tabs. This is where you manage all your scripts.
  7. Long-press a download link from the Mobile Downloads pin, copy the link, and paste it into Firefox's URL bar.

Before using any tools, you must install the Userscripts app.

  1. Open the App Store and search for Userscripts. Look for the </> icon by Justin Wasack. Tap GET to install.
  2. Open iPhone Settings → Apps → Safari → Extensions → Userscripts. (Or just search "Safari" in Settings.)
  3. Toggle Allow Extension to ON. Under Other Websites, select Allow.
  4. Tap the download link for a tool from the Mobile Downloads pin. Safari opens and shows the raw script text, or it may just pop up the script icon.
